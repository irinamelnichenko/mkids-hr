Необхідно підключити Google Apps Script

Система зберігає дані через Google Apps Script. Без нього клієнти не завантажаться. 📖 Як налаштувати? ↓
Кроки:
1. Відкрийте файл crm_script.js з репозиторію
2. Відкрийте конфігураційну таблицю Google Sheets
3. Extensions → Apps Script → вставте код з crm_script.js
4. Натисніть Run → setup() (один раз)
5. Deploy → New deployment → Web app
  • Execute as: Me
  • Who has access: Anyone
6. Скопіюйте URL деплойменту → вставте нижче
